# 亚洲世界杯长期预测模型

更新时间：2026-07-12

预测页是研究工具，不是 FIFA、AFC、足协、博彩公司或数据商的官方预测。页面回答的是“在当前公开信息与名额情景下，哪些亚洲球队更接近晋级线”，不是对未来名单的事实陈述。

## 覆盖和输出

- 候选母体为参加最近一届世界杯亚洲区预选赛的 46 支球队。
- 默认重点展示 2027 亚洲杯 24 强、概率前 20 名和中国队；完整榜单可展开。
- 预测届次为 2030、2034、2038、2042。
- 每支球队提供名额校准后的晋级概率、80% 区间、实力分档、数据覆盖率和主要信号。
- 2038、2042 只解释青训体系和留洋通道，不预测具体球员名单。

## 输入指标

`data/raw/forecast/model.json` 使用 0—100 的研究指数。指数由来源支持的国家级事实归一化，不等同于 FIFA 官方评分。

| 指标 | 含义 | 模型边界 |
| --- | --- | --- |
| `senior` | FIFA 排名语境、世预赛和亚洲杯成年表现 | 短期锚点，不单独决定远期结果 |
| `u17` / `u20` / `u23` | 各年龄级亚洲赛表现和相对强度 | U23 初始贡献高于 U20、U17 |
| `overseas` | 按联赛层级加权的一线队正式分钟 | 不计试训、梯队注册和无出场注册 |
| `transition` | 青年球员进入职业一线队的转化 | 公开资料不足时降低覆盖率 |
| `continuity` | 最近八年青年梯队表现稳定度 | 用于抑制单届爆发造成的高估 |
| `coverage` | 当前输入的可核覆盖程度 | 缺失数据向同层级历史中位数收缩，不视为零 |

当前海外指标按最近三赛季正式一线队分钟构造，使用 `log(1 + minutes)` 限制单个球员的支配作用；五大联赛顶级权重 1.0，其他欧洲顶级与五大次级 0.65，其他海外职业联赛 0.35。

## 模型流程

1. 历史面板使用 2002—2026 世界杯周期和 18 支主要竞争队，分别构造 4、8、12、16 年提前量；快照年份必须早于预测届次。
2. `SeniorModel` 和 `PipelineModel` 分别进行 L2 正则化逻辑回归。
3. 按世界杯周期滚动回测，以 Brier score 选择成年/梯队混合权重，并向提前量先验做收缩；成年权重随预测距离不得上升。如果梯队信号令回测恶化，可收缩到零。
4. 在 logit 空间移动统一截距，使所有 AFC 球队的概率之和等于配置的预期名额；2034 沙特固定为东道主 100%，其余球队共享 7.5 个预期席位。
5. 按世界杯周期进行 1,000 次固定种子 bootstrap，并叠加随提前量、数据覆盖率增大的输入误差；第 10 和第 90 百分位形成页面区间。

概率校准只保证“预期席位总量”一致，不模拟尚未公布的分组、赛程、主客场和洲际附加赛对手。

## 分档和对手

| 分档 | 概率 |
| --- | ---: |
| 稳定区 | ≥ 65% |
| 主要竞争区 | 35%—65% |
| 外围竞争区 | 15%—35% |
| 长期观察区 | < 15% |

中国主要对手按潜在实力分距离自动选择五队，优先包含排名线上、线下各两队。敏感度卡片只改变一项中国指标并重新校准全体概率，不属于正式预测。

## 名额假设与限制

- 2030、2038、2042 使用 AFC 8.5 个预期席位的研究情景，不能解释为 FIFA 已公布这些届次的资格赛规则。
- 2034 沙特为已确认东道主；当前假设自动席位包含在 AFC 8.5 席总量内。
- 历史训练队伍数、长期留洋分钟和旧届青年赛来源仍有限。页面公开训练样本数、回测指标和数据覆盖率，不承诺命中率。
- 原始国家指数需要在 FIFA 排名、世预赛、亚洲杯、AFC 青年赛或联赛分钟更新后复核；只更新文案不得改变预测结果。

## 生成和校验

```bash
npm run build-forecast
npm run validate-data
npm test
```

生成结果位于 `data/site/world-cup-forecast.json`。固定随机种子下重复构建必须完全一致；测试还会检查概率范围、名额总和、2034 东道主、未知远期东道主和区间宽度。
