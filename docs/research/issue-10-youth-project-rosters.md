# Issue #10：重点青训项目批次与逐名档案

核查日：2026-07-20。

本轮将根宝、足球小将、08 奥运之星、500 星、万达西班牙计划、鲁能足校和恒大足校统一为专题人物与批次关系两层。人物在专题内只有一个稳定 ID 和一条当前状态；同一人跨批次出现时不重复建人。批次分别保存公开人数、逐名人数、唯一人数、已核人数和待核人数。

- 足球小将 2014 苏州海选：官方最终 16 人名单，来源为[中国足球小将官方微博转载页](https://www.sina.cn/news/detail/4866099529648440.html)。
- 足球小将 2015 梯队：逐名 25 人，来源为[腾讯新闻 / 直播吧名单](https://news.qq.com/rain/a/20240319A01JFE00)。
- 足球小将 2016 曼努埃尔杯：逐名 12 人，来源为[直播吧名单](https://news.zhibo8.com/zuqiu/2025-10-31/690420b6197a5native.htm)。该名单只表示赛会关系。
- 08 奥运之星首批赴德名单：保持 27 人完整契约。
- 500 星计划：启动 24 人已按[同期出征报道的逐名表](https://sports.sina.cn/sa/2011-09-07/detail-ikknscsh8922277.d.html?vt=4)完整建档；累计回顾 41 人仍作为独立项目统计口径保存在 `program_metrics`，不为尚缺姓名的差额生成匿名人物。

## 统计边界

`prediction-only`、`partner-player`、`project-adjacent`、`tournament-only` 和 `needs-review` 不进入核心培养已验证人数。2034 杯球队赛果保存在 `event_records`。历史来源只能证明历史身份时，人物当前状态明确标记为 `needs-review`，不得把旧俱乐部沿用为当前归属。
