# Issue #11：UEFA Youth League 2013/14—2022/23 参赛队边界

核查日期：2026-07-19

## 统计口径

- 2013/14、2014/15 为试办期，只记录与成年欧冠小组赛相同的 32 支欧冠路径青年队。
- 2015/16 起赛事扩为 64 队，分别保存 32 支欧冠路径队和 32 支国内青年冠军路径队。
- 已举办赛季使用 `teams_by_path`，并将 `coverage.participating_teams` 标记为 `complete`。
- 2020/21 虽已完成 64 强抽签，但 UEFA 在首场比赛前取消赛事。因此只保存 `published_draw_teams_by_path`，覆盖状态为 `draw-published-not-played`，不计为实际参赛。

## 来源优先级

主来源为 UEFA 官方赛季历史页、当季赛制公告、参赛队确认公告、欧冠路径分组和国内冠军路径抽签公告。2020/21 以 UEFA 64 强抽签公告与取消公告共同限定边界。2013/14—2014/15 的球队范围同时遵循 UEFA “与成年欧冠小组赛组成相同”的官方赛制说明。

公开赛季汇总表仅用于交叉检查转写后的队名、路径归属和总数，不作为 `source_ids` 中的主事实来源。数据集采用现有英文队名风格，不把赞助商名、成年队完整法人名或不同语言拼写视为不同球队。

## 自动校验

- 九个已举办赛季的球队总数必须等于 `entrant_count`，且全季无重复。
- 2013/14、2014/15 只能出现 `champions_league`；2015/16 起两个路径必须各 32 队。
- `teams_by_path` 的键必须与赛季 `paths` 一致。
- 2020/21 不得出现 `teams_by_path`；已公布抽签名单必须是两路径各 32 队、共 64 队且无重复，同时所有比赛覆盖保持 `not-played`。

## 尚未扩展的边界

本轮不补 2013/14—2022/23 的全量逐场赛果，也不把报名后退出、比赛延期或球队名称沿革拆成独立实体。后续逐场采集仍应优先使用 UEFA match centre 或官方技术档案，并在球队名称发生变化时保留赛季当时的显示名与规范化映射。
