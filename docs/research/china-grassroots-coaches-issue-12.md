# Issue #12 中国基层青训教练扩展说明

更新时间：2026-07-19

本轮在首批9人基础上新增两个可由机构或中国足协资料闭环的梯队教练，并第一次把民间项目的出生年份批次从人物总档案中拆出。

## 已新增

- 山东泰山/鲁能足校：唐晓程，2025年山东泰山U15主教练快照。中国足协赛事报道同时确认球队、年龄段和职责。
- 恒大足校：中村雅昭，2023年恒大足校2008年龄段主教练。足校与中国足协资料相互印证；没有2026续任证据，因此不写成现任。
- 中国足球小将：在董路人物条目下新增2014批次职责，确认2026年比赛主教练与选材者范围；未公开具名的助教、守门员教练不填。

## 未完成边界

Issue 所称“各年龄段现任教练”是高变动名单，必须来自同一核查年度的具名机构资料。公开网页目前只能稳定确认部分赛事快照，不能把2021、2023和2025报道拼成2026组织架构。因此 watchlist 继续保留以下缺口：

- 山东泰山/鲁能足校 U10-U14、U17-U19 同年度主教练名单；
- 恒大足校 U9-U17 同年度主教练名单；
- 足球小将2015、2016等批次的长期教练、临时比赛教练和专项教练分工。

页面聚合继续由 loader 读取原始 JSON 并通过 `npm run build-data` 生成；生成文件不手工修改。
