# 王大雷、周海滨、冯潇霆、贾秀全海外经历核查

最近核查：2026-07-18

## 口径

- `试训`：俱乐部为评估球员安排的短期训练，不等于签约或注册。
- `跟训`：随队训练、学习或交流；如果俱乐部明确否认处于正式试训 / 转会评估阶段，就不写成签约前置程序。
- `正式留洋`：已经完成俱乐部合同或注册。即使成年一线队正式比赛为 0，也不能倒推成“仅试训”。
- 联赛、国内杯赛、洲际赛事和预备队赛事分别记录，不把各项赛事总出场写成联赛出场。
