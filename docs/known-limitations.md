# 已知限制

更新时间：2026-07-12

本项目是公开资料研究样本库，不是官方、全量或实时数据库。以下限制在使用页面、JSON 数据或研究结论时都应同时考虑。

## 非官方数据库

- 本项目不代表中国足协、AFC、FIFA、俱乐部、学校、联赛、经纪机构或第三方数据平台。
- 页面和 JSON 中的球员、教练、留洋数量只表示仓库已结构化并通过当前口径纳入的样本。
- “当前中国留洋样本”等数字不能解释为官方全量人数。

## 来源可能变化

- 俱乐部 squad、联赛注册页、赛事 PDF、新闻链接和数据库页面都可能改版、失效或更新。
- `verification.last_checked` 只表示仓库最近核查日期，不表示来源此后没有变化。
- 外部链接失效时，应按 `docs/source-policy.md` 和治理文档补替代来源，而不是直接删除证据链。

## 第三方资料可能冲突

- Transfermarkt、Wikipedia、Sofascore、FBref、WorldFootball、媒体报道和球探平台可能在姓名、生日、身高、位置、俱乐部或转会日期上互相冲突。
- 遇到冲突时，仓库用 `verification.status` 和 `verification.notes` 记录暂用口径。
- 单一非官方来源通常只能支撑 `provisional` 或 `needs-review`，不应覆盖官方注册字段。

## 青年球员资料更新滞后

- 青年队、学校、梯队、试训和短训信息公开不稳定，更新速度可能明显滞后。
- 很多小年龄段球员没有稳定俱乐部 profile 或联赛注册页。
- 未成年人相关信息只收录公开、必要、可复核的足球事实，不收录私人联系方式、家庭住址、证件信息或私人社媒材料。

## 当前注册不是实时转会

- `registration_club` 表示最近可核的当前有效注册，不是实时转会数据库。
- 未来生效转会、试训、传闻、商业选拔和训练邀请不改变当前注册字段。
- 当前留洋统计应区分当前海外注册、待生效、试训观察、回流和纯历史记录。

## 数据覆盖不均衡

- 中国、日本、韩国 U17/U23 已有较多种子数据，亚洲其他国家对照组仍在扩展。
- 男足 U20 世青赛与亚青赛的 1985—2025 届次概览、全部决赛圈球队及分组已经补齐，但不少旧届仍依赖 RSSSF/Wikipedia 等二级来源交叉，尚未逐届替换为完整官方技术报告；五大联赛长尾记录和亚洲教练扩展仍有缺口。
- 2027 两项 U20 赛事仅反映截至 2026-07-12 的官方状态。未公布的比赛日期、决赛圈参赛队和分组保持待定；AFC 资格赛抽签中的 44 队不等于已经取得决赛圈资格。
- 覆盖矩阵只说明仓库当前状态，不说明现实世界完整覆盖。

## 世界杯预测的边界

- 2030—2042 页面是实验性研究模型，不是官方资格结论、投注建议或确定名单。
- 概率以 AFC 8.5 席情景做总量校准；除已确认的 2034 沙特东道主外，未来名额、东道主、分组和赛制仍可能改变。
- 当前候选母体覆盖 46 队，但历史训练面板只覆盖 18 支主要竞争队，远期青年和留洋数据的公开程度也不均衡。
- 2038、2042 只适合比较体系信号。页面给出的 80% 区间不能覆盖战争、政策、归化规则、赛制改革、伤病和足协治理等结构性突变。

## 日韩体系页的边界

- 日韩体系页说明稳定制度和本站关联样本，不提供完整赛季赛果或全部注册球队。
- `competition_context_ids` 表示球员所处组织适用的竞赛环境，不等同于该球员在某赛季实际报名或出场。
- K League 俱乐部 U18 与合作高中名称可能随赛季调整；母俱乐部和合作学校映射按 `2026-07-12` 官方页面快照核查。
- Premier / Prince 队数、K League Junior 分组和比赛日期属于年度快照，不能作为永久规则引用。

## 生成数据不是独立事实源

- `data/site/**` 是由 `data/raw/**` 生成的页面数据，不应替代 raw 文件和原始来源。
- `storage/youth-football.sqlite` 是本地生成库，不提交、不发布，也不是权威数据源。
- GitHub Pages 部署不是数据快照；可追溯引用应固定到 commit、tag 或 release。

## 使用建议

- 引用重要结论时，同时引用原始来源链接和仓库 commit。
- 需要复用数据时，先阅读 `LICENSE`、`docs/scope.md` 和 `docs/api.md`。
- 发现错误、死链、隐私风险或口径争议时，优先使用 GitHub issue 模板提交。
